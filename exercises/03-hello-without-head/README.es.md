# `03` Hello sin Head

En el último ejercicio tuviste que agregar toda la estructura básica HTML de un sitio web.

## 📝 Instrucciones:

Haz lo mismo en este ejercicio, pero no incluyas la etiqueta HEAD.
Agrega al `<body>` un titulo `H1` con tu nombre.
